# Bank Churn Customer Analytics

## Overview
This project analyzes bank customer churn using exploratory data analysis, customer segmentation, and logistic regression. It identifies customers at risk of leaving and provides business insights for retention strategies.

## Objectives
- Understand customer behavior
- Segment customers into meaningful groups
- Predict churn probability
- Generate actionable business insights

## Dataset
Bank Customer Churn Dataset from Kaggle

## Tools and Libraries
- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Project Workflow
1. Data loading and inspection
2. Data cleaning
3. Exploratory data analysis
4. Customer segmentation using K-Means
5. Churn prediction using Logistic Regression
6. Model evaluation
7. Business insights and recommendations

## Key Results
- Built a churn prediction model using Logistic Regression
- Performed customer segmentation using K-Means clustering
- Evaluated performance using accuracy, confusion matrix, classification report, and ROC-AUC
- Derived business-oriented retention recommendations
